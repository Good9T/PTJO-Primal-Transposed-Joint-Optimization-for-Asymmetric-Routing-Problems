# PTJO
The source code of PTJO.<br>
We provide codes for two ARP variants<br>
- Asymmetric Traveler Salesman Problem (ATSP) <br>
- Asymmetric Capacitated Vehicle Routing Problem (ACVRP) <br>

# Basic Usage
To train a model
```bash
python train20.py
python train50.py
python train100.py
```
To test a model
```bash
python test20.py
python test50.py
python test100.py
```
All .py file contain parameters you can modify. <br>
